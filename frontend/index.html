AI Assistant - Modular Architecture

The application has been restructured to use a truly modular architecture!

Each module is now self-contained with:

  • Its own React components
  • Dedicated API interfaces
  • Package definition

Current modules:

  • Chat Module: Core conversational interface
  • Video Module: Video processing capabilities
  • Code Editor Module: Code editing and execution

Module files are located in their respective directories following the standardized structure:

modules/
  [module_name]_module/
    ui/
      react/
        components/
        api/
        module.js
